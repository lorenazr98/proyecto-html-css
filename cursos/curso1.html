Curso: Introducción al Desarrollo Web

Descripción

Este curso te brinda las bases para crear páginas web desde cero utilizando las tecnologías esenciales: HTML, CSS y JavaScript. Aprenderás de manera práctica cómo estructurar, diseñar y añadir interactividad a tus proyectos, sentando las bases para una carrera en desarrollo web.

Contenido del curso

  1. Fundamentos de HTML
    • Estructura básica de una página web.
    • Etiquetas principales (títulos, párrafos, enlaces, imágenes).
    • Creación de formularios sencillos
  2. Diseño con CSS
    • Introducción a estilos y selectores.
    • Diseño responsivo y uso de flexbox.
    • Personalización de colores, fuentes y diseño general.
  3. Interactividad con JavaScript
    • Conceptos básicos de programación.
    • Manejo del DOM para manipular elementos web.
    • Introducción a eventos (clicks, formularios interactivos).

Duración: 4 semanas.

Modalidad: Online, con sesiones prácticas y mentoría personalizada.

Nivel: Principiantes.

¿A quién está dirigido?

Este curso es ideal para personas sin experiencia previa que desean adentrarse en el mundo del desarrollo web y comenzar su aprendizaje en programación.
